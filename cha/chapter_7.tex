\section{网络的意义}
在我们生活的信息时代，网络对我们越来越重要。有些人在这里工作，有些人在这里购物，有些人在这里与久违的朋友互动。这是一个永远不会随着每一个新信息而改变的功能，本章内容希望让你熟悉如何在 Windows 11 中浏览网络。

\section{ISP}
 ISP 是互联网服务提供商的缩写。没有它，我们将无法访问互联网。所以我们需要一个 ISP 服务才能访问互联网。
 
 如果你想深入了解，那么可以看看下面的定义：
 
 \begin{definition}{ISP}
 	互联网服务供应商（英语：Internet Service Provider，简称ISP），又称因特网服务提供者、互联网服务提供商、网络服务供应商，即指提供互联网访问服务的公司。通常大型的电信公司都会兼任互联网服务提供商，一些 ISP 则独立于电信公司之外。
 	
 	互联网服务提供商透过固网早期的拨号连线、后来的 ADSL（非对称数字用户线路）、FTTx（光纤）、数据专线或是移动网络等方式提供互联网访问服务予客户。一些供应商还提供域名登记、网页托管和主机托管等服务。互联网服务提供商可以是商业营利性质的，也可以是由政府以公共财政支持，或国有资产。
 \end{definition}
 
 所以，说白了，这是一个网络提供商；再拿大白话讲，你得有运营商的宽带接入或者是移动数据网络，宽带接入包括以太网（Ethernet），无线局域网（WLAN）；移动数据网络包括蜂窝数据。
 
 如果你确定了你有上网的权利，那么下面就开始连接到你的网络并开始上网吧！
 
 \section{使用 Microsoft Edge 浏览网页}
 Microsoft Edge 是 Windows 11 操作系统的默认浏览器。你应该会发现它固定在系统的任务栏上。 只需左键单击打开它，就可以访问互联网上的各种网站。
